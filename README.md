# bilko.run

Personal site and AI tools by Bilko. Built with React, Fastify, Tailwind, and Gemini.

## Stack

- **Frontend**: React + Vite + Tailwind CSS v4
- **Backend**: Fastify + SQLite (better-sqlite3)
- **AI**: Gemini 2.0 Flash (REST API)
- **Payments**: Stripe (token-based)
- **Deploy**: Render

## Dev

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm start
```
